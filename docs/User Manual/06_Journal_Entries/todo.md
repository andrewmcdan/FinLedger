# Journal Entries

- Creating journal entries (manager and accountant) using chart of accounts.
- Multiple debits/credits per entry and balancing rules.
- Submit, approve, reject workflow and required rejection comments.
- Status views (pending, approved, rejected) with date filters.
- Search by account name, amount, or date.
- Error handling rules and messaging requirements.
- Attach source documents to entries and allowed file types.
- Notification to manager on submission.
