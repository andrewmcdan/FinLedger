# Sprint 5: Financial Ratios and Dashboard Feature List

Source PDF: `../SWE4713 - Application Domain - Sprint 5 - Financial Ratios and Dashboards Features List.pdf`

## All Users

1. Can view the financial ratios and dashboard on the landing page for all ratios where data is available in the solved problem, once successfully logged into the application.
2. Menu buttons found on other pages must also be accessible on the landing page depending on what each type of user is allowed to access: administrator, manager, and accountant.
3. A section of the landing page can be used to display important messages, such as journal entries waiting for approval.
4. Dashboards must display using three color codes: green for good, yellow for borderline or warning range, and red for items that need closer review. Research normal ranges for the different kinds of ratios; some examples were already provided in class.
5. The look and feel of the landing page must be consistent with the other pages in the application.
