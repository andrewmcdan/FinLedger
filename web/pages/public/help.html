Help

Get assistance and find answers to common questions about access, security, and accounting workflows.

Account Access
Login

Use your username and password from the login screen to access the app.

Request Access

Submit the request access form from the login page. New accounts require administrator approval.

Forgot Password

Use the "Forgot Password" flow to request a reset link and set a new password with your security answers.

Temporary Password Change

If your account requires a temporary password change, you will be redirected to the Change Password page after login.

Profile & Security
Update Profile Info

Edit your name, email, address, and profile image from the Profile page.

Change Password

Use the Change Password form on your Profile page.

Update Security Questions

Select new questions and answers on the Profile page.

Administrator Tools
User Approvals

Approve or reject new registrations from the Dashboard.

Create Users

Create users directly in the Dashboard and optionally upload a profile image.

Suspend & Reinstate Users

Suspend accounts for a date range and reinstate them when needed.

Reset User Password

Trigger a reset email for a user from the admin Dashboard.

Email a User

Send a direct email to a selected user using the Dashboard email form.

Data & Reports
Transactions Workspace

The Transactions page is organized into Journal, Journal Queue, and Ledger sections.

Journal and Journal Queue actions are available to manager and accountant roles. The administrator role sees the Ledger section only.

Journal Approval Access

Approve and Reject controls in Journal Queue are visible to manager users only.

If your role is accountant, you can prepare/submit entries and view queue status, but approval actions remain manager-only.

Reports and Trial Balance Link

Use Reports to review period-level summary information. Full report-generation workflows are still being completed.

From the Ledger section in Transactions, select "View Trial Balance as of this date" to jump to Reports with trial-balance query values prefilled.
